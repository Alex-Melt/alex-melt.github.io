# 思路

  • 获取请求头
  • 根据请求头分别从三个网站爬取 IP
  • 当 IP 池的数量达到设定数目时，停止爬取，等待后继续爬取
  • 将爬取的 IP 存入数据库，
  • 从数据库中获取 IP

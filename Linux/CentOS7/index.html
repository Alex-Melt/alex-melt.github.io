# IP

# 查看 IP

  • ip addr

若此时 IP 显示不出来，则进入设置

  • 进入根目录 cd /
  • 进入配置 IP 所在的目录 cd etc/sysconfig/network-scripts
  • 编辑 vi ipcfg-en33
  • 使用 i 进入编辑，
  • 修改最后一项为 yes
  • 编辑完成后 esc 退出，使用 :wq 进行保存
  • 重启网络检查是否生效 systemctl restart network

# ifconfig 命令找不到

  • 使用 yum search ifconfig
  • yun install net -tools.x86_64

# 检测是否连接网络

  • ping www.baidu.com

# vim 编辑器

vim 是 vi 编辑器的增强版，有三种模式：一般模式，编辑模式，命令模式

  • 一般模式：正常的浏览模式，不做任何操作
  • 编辑模式：可以对当前模式进行模式进行修改
  • 命令模式：通过输入一些命令来完成对文本的一些操作

# 安装 vim 编辑器

  • yun -y install vim

查看是否安装 vim：

  • rpm -qa|grep vim

# vim 常用命令

  • 创建文件：vim 文件名称，如果文件存在则直接进入文件
  • 进入编辑模式：按下 i 或者 o
  • 退出编辑模式：按下 esc 键
  • 进入命令模式：输入：或者 /
  • 退出命令模式：按下 esc 键
  • 退回到 linux 命令行：
    • 保存对文件的修改并退出 :wq
    • 不保存对文件的修改 :q
    • 不保存对文件的修改并强制退出 :q!
  • 撤销对文件的编辑，等同于 ctrl+z u
    • 先保证是在一般模式下，
  • 拷贝当前行，然后粘贴
    • 在一般模式下，对某行复制 按住两次 y，粘贴的位置按下 p
  • 拷贝多行
    • 在一般模式下，在首行按下需要拷贝的行数，复制 按住两次 y，粘贴的位置按下 p
  • 删除当前行
    • 在一般模式下，按下两次 d
  • 删除多行：
    • 按下行数加两次 d

# 防火墙状态

  • 查看防火墙状态：systemctl status firewalld
  • 设置开机启用防火墙：systemctl enable firewalld.service
  • 设置开机禁用防火墙：systemctl disable firewalld.service
  • 开启防火墙：systemctl start firewalld
  • 关闭防火墙：systemctl stop firewalld

# 环境变量

  • 查看所有环境变量
    • env
  • 环境变量生效
    • source /etc/profile
